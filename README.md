📚 Exam Notes AI

An AI-powered web application that helps students generate smart exam notes with charts, diagrams, and downloadable content. It also supports secure Google authentication and Razorpay payments for premium features.

🚀 Features
🔐 Google Authentication (OAuth 2.0)
🤖 AI-based Notes Generation
📊 Charts & Diagram Support
📥 Download Notes (PDF)
💳 Razorpay Payment Integration
🧑‍💻 User-specific Dashboard
🛠️ Tech Stack

Frontend

React.js
Tailwind CSS

Backend

Node.js
Express.js

Database

MongoDB

Authentication

Google OAuth 2.0

Payments

Razorpay

AI

OpenAI API
